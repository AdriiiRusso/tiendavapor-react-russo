# Tienda Vapor
Proyecto para Coderhouse en React.


## Puesta a punto
Para tener la aplicación funcionando hay que realizar:


### `git clone [URL repo]`
Con el fin de clonar el repo.

### `npm install`
Para instalar las dependencias y el server.

### `npm start`
Para inicializar el servidor y ver la app.
